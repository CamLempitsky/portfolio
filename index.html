Controls

Use keys W, A, S, D to move

Left click to use selected item

Select hotbar slots with the number keys

Press Space while moving to use the dash ability

Press H to use healing potion (if usable)


What to do

Defeat enemies to gain items and boosts that increase Doug's strength.

There exists a number of bosses that are advisable to fight in a certain order:

First you should defeat the bear, it roams the forest, hungry. Tales tell that if
one defeats it they may be able to possess a tool which allows them to attack
from afar.

The final boss is known to fly in the skies and breathe fire. It must be defeated
Doug to escape this world and return home. Does he have what it takes?
